Title: 2026年6月稳定版更新

----------------------

2026年6月稳定版已发布！

本月公告内容有1.4.5更新进度、月内新模组展示、稳定版更新日志。

[h1]尚未支持泰拉瑞亚1.4.5[/h1]

如上次公告所说，让 tModLoader 跟上泰拉瑞亚1.4.5（更大更石）的大量新内容、新机制要些时间。这些更新连带模组制作社区想要的大改动，实在得花好几个月。

[h2]泰拉瑞亚1.4.5支持进度[/h2]

这个月的更新卓有成效，我们收到了许多资深模组制作者的贡献。十分感谢助力的社区成员。tModLoader 团队将继续与模组制作者共同推进1.4.5更新。

项目仍有很多待办，如果你想为1.4.5更新贡献代码，看[url=https://github.com/tModLoader/tModLoader/issues/5070]1.4.5更新信息[/url]了解更新细节和任务清单。用户也可以按此页面说明的方法加入“1.4.5-dev”测试版游戏，帮忙找漏洞，反馈问题。不过别期待这个版本能正常游玩。

即使不能亲自贡献代码，作为模组制作者，你也可以加入 Discord，与 tModLoader 团队和贡献者讨论 API 的不足之处。我们希望借这次重大更新拔除常规月度更新不便修改的问题。

社区贡献内容一般先经过 Discord 讨论，再搬到 GitHub 上准备实现。

[h2]关于泰拉瑞亚1.4.5支持的常见问题[/h2]
[b]问：“我的模组到1.4.5也都马上能玩吧？”[/b]
答：泰拉瑞亚1.4.5更新量很大，所以我们不敢肯定。不过，模组制作者有充足机会在1.4.5正式上线前更新模组。只要你玩的模组有人维护，到时大概能正常游玩。

[b]问：“那1.4.4呢？”[/b]
答：就像1.3和1.4.3一样，更新后1.4.4会作为旧版本供大家游玩。

[b]问：“不更新1.4.4了吗？”[/b]
答：我们会继续维护1.4.4，确保功能正常。何时停止添加内容，还没有确切日期；直到1.4.5可供模组制作者试用，我们会照常更新1.4.4。

[b]问：“1.4.4的人物和世界搬到1.4.5会炸吗？”[/b]
答：tModLoader 视保障玩家体验为己任。我们尽力确保新旧兼容，届时也会鼓励模组制作者考虑兼容性。
